Question-1:What are the different ways to select an element in the DOM?
There are 6 different ways to select an element in the dom.
  1. 1.getElementById - Selects an element by its ID.
  2. 2.getElementsByClassName - Selects elements by their class name.
  3. 3.getElementsByTagName - Selects elements by their tag name.
  4. 4.querySelector - Selects the first element that matches the CSS selector.
  5. 5.querySelectorAll - Selects all elements that match the CSS selector.
Question-2:What are the difference between innerHTML,innerText and textContent?
  • innerHTML - Gets or sets HTML content, including tags.
  • innerText - Gets or sets only visible text, ignoring hidden elements.
  • textContent - Gets or sets all text, including hidden text, without parsing HTML.
Question-3:What is an evnet delegation in DOM?
Event delegation is a technique where a parent element handles events for its child elements using a single event listener.
Question-4:What is an evnet bubbling in DOM?
Event bubbling is a process where an event starts from the target element and propagates up to its parent elements
Question-5: How do you create, add, and remove elements using JavaScript?
  • createElement: Creates a new element.
  • appendChild: Adds an element as the last child of a parent.
  • remove: Removes an element from the DOM.
